# コミュニケーションサイト
コミュニケーションサイトを利用したシンプルなポータルサイトテンプレートです。  
弊ブログでも紹介しております。  
[鍋綿ブログ](https://www.micknabewata.com/entry/sharepoint/communicationSiteTemplate)  

# 機能一覧
- ナビゲーション
    - ホーム
    - ニュース
    - イベント
    - ドキュメント
- ホームページ
    - ニュース
    - リンク集
    - イベント
    - ドキュメント
    - 天気
    - ハブのニュース
    - 人気のコンテンツ
- フッター
